## Objective
This project is a simple Password Generator application built using Python and Tkinter.
It generates random passwords based on user-selected criteria.

## Tools Used
- Python
- Tkinter
- random module
- string module

## Features
- GUI-based interface
- Custom password length
- Option to include letters
- Option to include numbers
- Option to include symbols
- Input validation
- Copy to clipboard feature
- Clear button

## How It Works
The user enters the password length and selects the desired character types.
The program then generates a random password accordingly.

## How to Run
Run the program using:

python password_generator.py

## Outcome
The application successfully generates secure random passwords based on user input.
